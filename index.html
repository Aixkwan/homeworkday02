git提交代码的基本操作

git操作,可通过命令面板创建文件夹: mkdir css(创建一个名为css的文件夹)

git init 初始化本地git仓库

git status 查看文件列表(若文件存在,则文件名红色,若文件存在并添加到暂存区之后文件名绿色)

git add . 添加所有文件到本地暂存区

git commit -m "第一次提交项目文件" 将文件提交到本地git仓库

git branch main 创建主分支 (可跳过)

git remote add origin *** (远程仓库地址)

git push -u origin master 推送文件到远程git仓库主分支上

第二次提交文件时(修改项目或新增)

git add . 添加所有文件到本地暂存区

git commit -m "第二次提交项目文件(修改之后的版本)"

直接提交文件到远程git仓库昂主分支上
